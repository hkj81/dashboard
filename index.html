🔑 GitHub 토큰 설정

팀원 공유를 위해 GitHub Personal Access Token이 필요합니다.

1. github.com/settings/tokens/new 접속
2. Note: 아무 이름 입력 → Expiration: No expiration
3. repo 체크 → Generate token
4. 생성된 토큰(ghp_로 시작) 복사 후 아래 입력
